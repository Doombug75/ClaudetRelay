INPUT — Locked Reference Folder
================================
Files in this folder are READ-ONLY for all AI participants.
AI can read these files but cannot write, delete, or modify them.

Use this folder for:
  • Finished chapters or sections you want to lock from further AI edits
  • Reference material: style guides, research notes, world bibles
  • Source documents the AI should treat as canon

To promote a finished file from OUTPUT to INPUT:
  Open the 📂 Files panel → OUTPUT section → click "🔒 → INPUT"
